# InteligenciaArtificial_Grupo13_5k09_2025
  Nuestro Sistema Experto se centra principalmente en el monitoreo y logista de la "Gestión de Recolección de Residuos", cuyo sistema inteligente debe ser capaz de detectar automáticamente contenedores llenos fuera del cronograma planificado y asignar en tiempo real el camión óptimo para resolver la situación, equilibrando eficiencia operativa, tiempos de respuesta y continuidad del servicio.
